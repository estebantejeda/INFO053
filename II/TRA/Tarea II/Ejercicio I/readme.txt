El código Main.cpp es un algoritmo el cual realiza diversos tipos de ordenamientos en un arreglo, con el fin de calcular el tiempo que demora cada uno y así poder ver la eficiencia de estos.

Se crearon tres métodos de ordenamiento, además de utilizar uno ya existente en la librería std c++11, cada uno con sus respectivos comentarios que explican a grosso modo el funcionamiento de estos.

El programa se debe ejecutar como "./main n", donde n representa el tamaño del arreglo. 
"n" debe ser un número positivo (n > 0).

Una vez realizado esto, el programa ordenará de distintas maneras el arreglo, imprimiendo en pantalla el arreglo original (Rellenado con números aleatorios) para luego mostrar más abajo el mismo arreglo ordenado indicando el método utilzado.

Además de lo anterior, el algoritmo imprimirá en pantalla el tiempo que tomó cada método en ser utilizado, con el fin de que el usuario pueda sacar sus propias conclusiones.
