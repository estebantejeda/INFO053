El código Main.cpp es un algoritmo el cual realiza la búsqueda de un número en tres estructuras diferentes rellendas con números aleatorios con el fin de comparar de manera empírica el tiempo que toma realizar la búsqueda.

El programa se debe ejecutar como "./main n", donde n representa el tamaño del arreglo. 
"n" debe ser un número positivo (n > 0).

Una vez realizado esto, el programa creará tres estructuras distintas (Arreglo Ordenado, Árbol AVL y Tabla de Hash) las cuales rellenará con números aleatorios <= 500. Se tomará un número aleatorio y se intentará buscar en cada tipo de estructura. El programa aparte de contar el tiempo en que se demora la búsqueda, también contará cuántas veces se encuentra ese número.
